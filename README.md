# THRIDHA - Premium Interior Solutions

## Project Info

**URL**: [THRIDHA Interior Solutions](https://THRIDHAinteriors.com/)

## Getting Started

To work locally with this project:

1. Clone the repository
2. Install dependencies:
   ```sh
   npm install
   ```
3. Start the development server:
   ```sh
   npm run dev
   ```

## Technologies Used

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
